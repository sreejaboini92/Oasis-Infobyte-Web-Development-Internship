# Task 1 - Landing Page

## Description

This project is a VPN-themed Landing Page developed using HTML and CSS as part of the Oasis Infobyte Web Development Internship.

The landing page is designed to promote a VPN service by highlighting its features, secure browsing experience, and user-friendly interface.

## Features

* Responsive Navigation Bar
* Hero Section with Call-to-Action Button
* Features Section with Information Cards
* Hover Effects on Buttons and Cards
* Modern UI Design with Shadows and Rounded Corners
* Clean and Structured Layout

## Technologies Used

* HTML5
* CSS3

## Project Structure

* Navbar
* Hero Section
* Features Section
* Footer (if added)
