# Task 3 - Temperature Converter Website

## Description

This is a Temperature Converter Website developed using HTML, CSS, and JavaScript as part of the Oasis Infobyte Web Development Internship.

The application allows users to enter a temperature value, select a unit, and convert it to another temperature scale.

## Features

* Temperature Input Field
* Unit Selection (Celsius/Fahrenheit)
* Convert Button
* Instant Result Display
* Input Validation
* User-Friendly Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
